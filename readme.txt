開発頓挫日：2022/12/14（Wed）

面談日程を組む際に、生徒一人ひとりにLINEで日程候補を送る形式だと
    ・文面を作るのがめんどくさい
    ・生徒の希望日程が被るケースがある
    ・希望日程が被った際に改めて日程を提示する必要がある
という課題があった。
そこで、あらかじめ候補日程を列挙し、生徒に希望日程に申込みをしてもらうことで、上記の課題を解決しようとした。
が、2022.12.14にモチベーションが消え、開発は頓挫。面倒になって実装をやめた機能
    ・ユーザー毎にイベントを管理する機能（ユーザー登録、ログイン画面など）
    ・生徒がスマホで希望日程に申し込む画面の作成
    ・イベント一覧や日程一覧のスケジュール機能
    ・イベントの編集画面（日程編集画面と同様にモーダルウィンドウで実装するつもりだった）
    ・各ボタンにポイントを当てた際にポップアップで軽い説明が表示される機能
この開発自体は頓挫したが、（少なくとも簡単なwebアプリ程度なら）未経験の言語でもGoogleとYoutubeでそれなりのものは作れると思った。
実際、phpは何も学習せずこのアプリが作れたので、文法などの基礎を３日くらいで勉強すればもっと綺麗なコードが書けると思った。
